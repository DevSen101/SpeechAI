## 📌 Overview

SpeechAI is a **voice-enabled AI assistant** built using the **Google Gemma 2B** language model. It allows users to interact using **text or voice input** and receive responses in both **text and audio format**.

This project demonstrates the integration of:

* Large Language Models (LLMs)
* Speech Recognition
* Text-to-Speech (TTS)
* Interactive Web UI using Gradio

---

## 🚀 Features

* 🎙️ Accepts **voice and text input**
* 🧠 Powered by **Gemma 2B (LLM)**
* 🔊 Converts responses into **speech (gTTS)**
* ⚡ Supports **GPU acceleration (CUDA)**
* 🌐 Interactive UI using **Gradio**
* 🔄 End-to-end pipeline: Speech → Text → AI → Speech

---

## 🛠️ Tech Stack

* **Python**
* **PyTorch**
* **Hugging Face Transformers**
* **Gradio**
* **SpeechRecognition**
* **gTTS (Google Text-to-Speech)**

---

## ⚙️ How It Works

1. User provides input (text or voice)
2. Voice input is converted to text using SpeechRecognition
3. Text is tokenized and passed to the Gemma 2B model
4. Model generates a response
5. Response is converted into audio using gTTS
6. Output is displayed as text + audio
